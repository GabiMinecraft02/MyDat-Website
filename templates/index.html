Help

Commands
mydat -e "file" "encrypt key" # Encrypts the file with the specified key.
mydat -d "file" "decrypt key" # Decrypts the file with the specified key.
mydat -g "bits number" # Generates a new key with the specified number of bits.
mydat -b "folder" "file" # Generates a new 0% compressed file with the specified folder and file.
mydat -i "file" "folder" # Extracts the contents of the specified file to the specified folder.
